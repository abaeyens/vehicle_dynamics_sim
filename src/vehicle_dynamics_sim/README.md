# vehicle_dynamics_sim

ROS 2 package for high-fidelity vehicle dynamics simulation
with realistic actuator modeling.

## Documentation

**Complete documentation is available in the
[repository README](../../README.md).**

This package provides the core vehicle dynamics simulator
with support for multiple vehicle types
(bicycle/Ackermann, differential drive),
realistic actuator dynamics (delays, filtering, limits),
and optional localization simulation.
